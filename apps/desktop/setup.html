Welcome to mITyStudio

Pick a folder for your music workspace. Your projects, SoundFonts, samples, voice recordings and exports will live there — plain files you own, never locked in.

You can add SoundFonts (.sf2) and samples to this folder at any time — the studio picks them up with one rescan. API keys for the chat assistant are set later inside the app (Settings), or read from environment variables automatically.
